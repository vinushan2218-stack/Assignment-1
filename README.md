# Image Preview Test Automation

This project uses Python and Playwright to test whether a PNG file preview is displayed after uploading an image to a website.

## Project Files

- `image_preview_test.py` - Main automation script
- `sample.png` - Test PNG image file
- `results/` - Stores screenshots
- `execution_results.csv` - Stores test results

## Requirements

- Python 3.11 or 3.12
- Google Chrome or Playwright Chromium
- Playwright
